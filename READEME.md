# 📚 MyVoca: 단어 암기와 복습을 돕는 개인 단어장

<br>

**MyVoca**는 개인 단어 암기 및 학습 기록 관리를 위한 웹 서비스입니다.

사용자가 직접 학습 단어를 등록하면, 시스템은 **단어를 뭉치**로 구성하고 **기억 곡선**의 원리를 활용하여 가장 효율적인 시점에 복습을 유도합니다.<br>
이 서비스를 통해 사용자는 학습 진행 상황을 체계적으로 추적하고 학습 효율을 극대화할 수 있습니다.
<br><br>

## Features

MyVoca는 다음의 주요 학습 기능을 제공합니다.
<br><br>

1.  간편한 단어 관리:

    - 사용자가 단어와 뜻을 입력하고 저장합니다.
    - 시스템은 입력된 단어를 일정 개수의 뭉치로 자동 분리합니다.
    - 사용자는 뭉치의 개수나 포함될 단어 목록을 직접 지정할 수 있으며, 설정이 없을 경우 시스템이 자동으로 구성합니다.

2.  다양한 암기 형태:

    - 플래시카드: 단어(앞면)와 뜻(뒷면)을 전환하는 기본 학습 방식입니다.
    - 객관식 퀴즈: 복수의 선택지는 보다 쉬운 암기 난이도를 제공합니다.

3.  직관적인 UI:

    - 시스템이 뭉치별 학습 스케줄을 요일별로 권장하고, 정해진 기간 내 학습 완료 여부를 달력 형태로 시각화하여 제공합니다.

4.  편리한 복습:
    - 시스템이 뭉치 및 단어별 암기 상태를 파악하고, 최적의 복습 시점에 알림을 제공합니다.
